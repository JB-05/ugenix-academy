# UgeniX Academy

Professional training academy website built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `/app` - Next.js App Router pages and layouts
- `/components` - Reusable React components (Header, Footer)
- `/public` - Static assets

## Pages

- `/` - Home page with hero, course highlights, and information sections
- `/courses/prompt-engineering` - Course detail page
- `/register` - Registration form
- `/admin` - Admin dashboard for managing registrations

## Design Philosophy

The design emphasizes:
- Human-crafted, intentional spacing
- Professional typography
- Custom color palette (deep slate, muted blue/teal, soft neutrals)
- Content-first, minimal aesthetic
- Mobile-first responsive design
